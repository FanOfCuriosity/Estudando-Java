# Studying Java

# **Context**

## This project brings together the problem statements and solutions I have completed throughout my journey of learning the Java programming language, as well as its frameworks.

## I use the **Big Java** book to study the theory and **ChatGPT** to create the exercises.

## For each chapter of the book, I create a separate repository containing only the exercises for that chapter.

* ## The exercises in each chapter cover everything I have learned so far, including the current chapter's content. For example, after finishing the theory in Chapter 7, I solve exercises that require knowledge from Chapters 1 through 7.

# **Objective**

* ## To make my progress in Java public and tangible. Rather than simply claiming that I know Java, I want to provide evidence of my skills through this repository. Anyone can browse it and follow my improvement over time—not only in the language itself, but also in libraries, frameworks, code organization, and clarity, all of which are essential skills in the software development industry.
